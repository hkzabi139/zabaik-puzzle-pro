# 🧩 Zabaik Puzzle Pro - Image Puzzle Game

A professional, responsive, and interactive Image Puzzle Game built with modern web technologies. This project was developed as part of **Task #9** for the **Developer Super League**.

## ✨ Features
- **Dynamic Image Splitting:** Upload any image from your device, and the system automatically converts it into a puzzle.
- **Adjustable Difficulty:** Choose between different grid sizes (3x3 Easy, 4x4 Hard).
- **Responsive Design:** Optimized for both Desktop and Mobile devices using CSS Flexbox/Grid.
- **Drag & Drop Logic:** Seamless drag-and-drop experience for Desktop and intuitive Touch events for Mobile.
- **Verification System:** A "Confirm you're not a robot" style gameplay.
- **SweetAlert2 Integration:** Beautiful, modern pop-up notifications upon puzzle completion.
- **Reference Helper:** A real-time reference image to assist users during gameplay.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure and File API for image uploads.
- **CSS3:** Custom properties (Variables), Grid, Flexbox, and Media Queries for responsiveness.
- **JavaScript (ES6+):** Core logic for image splitting, shuffling, and drag-and-drop events.
- **SweetAlert2:** For premium UI notifications.
